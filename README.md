# Aleo Hackathon Landing Page

Next.js + Tailwind CSS single-page site for promoting the Aleo x OpenBuild Hackathon.

## Tech Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4

## Development

```bash
npm run dev
```

Open http://localhost:3000.

## Production Check

```bash
npm run lint
npm run build
```

## Content Notes

- Main content lives in `src/app/page.tsx`.
- Global design tokens live in `src/app/globals.css`.
- The two QR panels in the报名 section are visual placeholders. Replace them with real QR images or links when the official registration and Hacker group assets are ready.
